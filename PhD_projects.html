My work has focused so far on reconstructing clonal populations from whole genome sequencing data. This has lead me to develop QuantumClone, which aims to reconstruct clonal populations by statistical inference. I have also been interested in how to select the best variants to do the clustering.

R packages :

  • QuantumClone
    Can be found either on CRAN or on GitHub

    QuantumClone has been designed to reconstruct clonal populations from High Throughput Sequencing data (whole exome or whole genome sequencing). It reconstructs clones based on the probability to belong to a cluster relying on a binomial probability knowing the cellular prevalence of the clone, the number of reads mapping this position and the number of alternative reads. For variants in hyper-diploid regions, the number of copies is selected so that it maximizes the probability to belong to a clone. It has been used to infer clonal populations in Clonal assessment of functional mutations in cancer based on a genotype-aware method for clonal reconstruction, under review

  • ACSNMineR
    Can also be found on CRAN or GitHub

    ACSNMineR has been designed to facilitate computation of enrichment or depletion of pathways based on knowledge from the Atlas of Cancer Signalling Network. It also eases imports of GMT files, to compute enrichment on user specific entries. It also integrates easy syntax for plots based on ggplot2. A more detailed description and examples can be found in Calculating Biological Module Enrichment or Depletion and Visualizing Data on Large-scale Molecular Maps with ACSNMineR and RNaviCell packages, The R Journal, 2016

  • DevGRaph
    DevGRaph is a package that is intended to ease package development. It can go through R scripts and display function interactions. In addition, it can display functions imported from other packages, which can be useful to list imports or when checking conflicts between imported packages. It also provides easy syntax for plots.
